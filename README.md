## Code and Sample Data for Computational Materials Science Submission

**DOI (this repository):** 10.5281/zenodo.XXXXXXX

## Overview

This repository contains the training code and a sample dataset for the gatekeeper-constrained dual-stage Random Forest (RF) surrogate model described in the paper.

The surrogate emulates SRIM-2013 Binary Collision Approximation (BCA) physics and simultaneously predicts **nine output quantities** for ion implantation across Si, SiC, and GaAs substrates:

| Output | Symbol | Unit |
|--------|--------|------|
| Projected range | Rp | Å |
| Longitudinal straggle | ΔRp | Å |
| Lateral range | Rlat | Å |
| Lateral straggle | ΔRlat | Å |
| Radial range | Rrad | Å |
| Radial straggle | ΔRrad | Å |
| Vacancy density | Vvac | vacancies/ion |
| Backscattered ions | Nback | count |
| Transmitted ions | Ntrans | count |

## Repository Contents

```
.
├── train_model.py        Training script (Stage 1 classifier + Stage 2 regressors)
├── predict.py            Inference script (interactive, CLI, and batch modes)
├── sample_database.csv   500-row sample dataset (see note below)
├── requirements.txt      Python package dependencies
└── README.md             This file
```

> **Note on sample_database.csv:**  
> This file contains **500 representative rows** selected from the full 5,628-simulation training corpus. Rows corresponding to the 29 experimental validation conditions reported in Table 5 of the paper are included. The sample dataset allows verification of the code pipeline and structure, but **will not reproduce the paper's quantitative results** (R² ≥ 0.92, etc.), which require the complete training set. The full dataset can be generated by running SRIM-2013 with the parameter grid described in Section 3 of the paper.

## Architecture

```
Input (5 parameters)
  └─► Stage 1: Gatekeeper RF Classifier
        ├─ Class 0 (Transmitted) ──► Rp = 0  (direct return)
        └─ Class 1 (Stopped)     ──► Stage 2: RF Regressors
                                        ├─ Geometry Regressor  (Rp, ΔRp, Rlat, ΔRlat, Rrad, ΔRrad)
                                        └─ Damage Regressor    (Vvac, Nback, Ntrans)
```

**Input parameters:**
- `substrate` ∈ {Si, SiC, GaAs}
- `ion` ∈ {B, Mg, P, Ar, As}
- `energy_keV` — implantation energy (10–10,000 keV)
- `angle_deg` — tilt angle (0°–89.9°)
- `thickness_A` — substrate layer thickness (100–10,000 Å)

## Usage

### Step 1 — Train the models

```bash
python train_model.py
```

This trains all three model components using 10-fold cross-validation and saves them to `models/`:
- `models/gatekeeper_classifier.pkl`
- `models/geometry_regressor.pkl`
- `models/damage_regressor.pkl`

Optional arguments:
```bash
python train_model.py --data sample_database.csv --out models/ --cv 10
```

**Expected output (sample data, 500 rows):**
```
Stage 1 — Gatekeeper Classifier
  10-fold CV  Accuracy: ~0.85  (paper: 0.919 on full 5,628-row dataset)
  10-fold CV  F1-score: ~0.88  (paper: 0.951)

Stage 2 — Regressors
  Geometry regressor  10-fold CV  R²: ~0.75  (paper: 0.944–0.977)
  Damage regressor    10-fold CV  R²: ~0.70  (paper: 0.974–0.985)
```

> Lower performance on the sample dataset is expected — see note on `sample_database.csv`.

### Step 2 — Run predictions

**Interactive mode:**
```bash
python predict.py
```

**Single prediction (CLI):**
```bash
python predict.py --substrate SiC --ion B --energy 300 --angle 7 --thickness 5000
```

**Batch mode:**
```bash
python predict.py --batch my_conditions.csv --out predictions.csv
```

The batch CSV should have columns: `substrate, ion, energy_keV, angle_deg, thickness_A`

## Dataset Description

`sample_database.csv` — Column definitions:

| Column | Type | Description |
|--------|------|-------------|
| substrate | str | Target material: Si, SiC, or GaAs |
| ion | str | Implanted ion: B, Mg, P, Ar, or As |
| energy_keV | float | Implantation energy (keV) |
| angle_deg | float | Beam tilt angle (degrees) |
| thickness_A | float | Substrate layer thickness (Å) |
| Rp_A | float | Projected range (Å) |
| dRp_A | float | Longitudinal straggle (Å) |
| lateral_range_A | float | Lateral range (Å) |
| lateral_straggle_A | float | Lateral straggle (Å) |
| radial_range_A | float | Radial range (Å) |
| radial_straggle_A | float | Radial straggle (Å) |
| vacancies_per_ion | float | Displacement damage vacancies per ion |
| backscattered | float | Number of backscattered ions (out of 10,000) |
| transmitted | float | Number of transmitted ions (out of 10,000) |
| is_profile | int | 1 = stopped (Rp ≥ 10 Å); 0 = transmitted |

All SRIM simulations used 10,000 ions per run in TRIM full-damage mode with amorphous single-layer targets.
