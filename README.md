## Overview

This repository contains the training code and a representative sample dataset for the gatekeeper-constrained dual-stage Random Forest surrogate model described in the paper.

The surrogate emulates SRIM-2013 Binary Collision Approximation (BCA) physics and simultaneously predicts **nine output quantities** for ion implantation across three semiconductor substrates (Si, SiC, GaAs) and five ion species (B, Mg, P, Ar, As):

| Output | Symbol | Unit |
|--------|--------|------|
| Projected range | Rp | Å |
| Longitudinal straggle | ΔRp | Å |
| Lateral range | Rlat | Å |
| Lateral straggle | ΔRlat | Å |
| Radial range | Rrad | Å |
| Radial straggle | ΔRrad | Å |
| Vacancy density | Vvac | vacancies/ion |
| Backscattered ions | Nback | count |
| Transmitted ions | Ntrans | count |

**Key results:**
- Mean R² = 0.911 under ten-fold cross-validation
- 715× mean speedup over SRIM on a Core i5 processor
- 8.3% mean absolute percentage error against 28 published experimental conditions

└── README.md             This file

> **Note on sample_database.csv:**  
> This file contains **500 representative rows** from the full 5,611-simulation training corpus. It allows verification of the code pipeline and data structure, but **will not reproduce the paper's quantitative results** (R² ≥ 0.911), which require the complete dataset. The full training corpus can be regenerated by running SRIM-2013 with the parameter grid described in Section 2 of the paper.

## Model Architecture

```
Input: (substrate, ion, energy_keV, angle_deg, thickness_A)
  │
  └─► Stage 1: Gatekeeper RF Classifier (n_estimators=100)
        │
        ├─ Class 0 (Transmitted) ──► Rp = ΔRp = Vvac = 0  [direct return]
        │
        └─ Class 1 (Stopped) ──► Stage 2: Parallel RF Regressors
                                    │
                                    ├─ Geometry Regressor (n_estimators=150)
                                    │     Rp, ΔRp, Rlat, ΔRlat, Rrad, ΔRrad
                                    │
                                    └─ Damage Regressor (n_estimators=100)
                                          Vvac, Nback, Ntrans
```

**Input parameter space:**
- `substrate` ∈ {Si, SiC, GaAs}
- `ion` ∈ {B, Mg, P, Ar, As}
- `energy_keV` — implantation energy: 10–10,000 keV
- `angle_deg` — tilt angle: 0°–89.9°
- `thickness_A` — substrate layer thickness: 1,000–10,000 Å

## Usage

### Step 1 — Train the models

```bash
python train_model.py
```

Trains all three model components using 10-fold cross-validation and saves them to `models/`:
- `models/gatekeeper_classifier.pkl`
- `models/geometry_regressor.pkl`
- `models/damage_regressor.pkl`

Optional arguments:
```bash
python train_model.py --data sample_database.csv --out models/ --cv 10
```

**Expected output on sample data (500 rows):**
```
Stage 1 — Gatekeeper Classifier
  10-fold CV  Accuracy : ~0.85  (paper: 0.924 on full 5,611-row corpus)
  10-fold CV  F1-score : ~0.88  (paper: 0.952)

Stage 2 — Regressors
  Geometry regressor  10-fold CV  R² : ~0.75  (paper: 0.944–0.976)
  Damage regressor    10-fold CV  R² : ~0.70  (paper: 0.973–0.998)
```

> Lower performance on the sample dataset is expected — the full training corpus is required to reproduce the paper's results.

---

### Step 2 — Run predictions

**Interactive mode:**
```bash
python predict.py
```

**Single prediction (CLI):**
```bash
python predict.py --substrate SiC --ion B --energy 300 --angle 7 --thickness 5000
```

**Batch mode:**
```bash
python predict.py --batch my_conditions.csv --out predictions.csv
```

Batch CSV must have columns: `substrate, ion, energy_keV, angle_deg, thickness_A`

## Dataset Description

`sample_database.csv` — Column definitions:

| Column | Type | Description |
|--------|------|-------------|
| substrate | str | Target material: Si, SiC, or GaAs |
| ion | str | Implanted ion: B, Mg, P, Ar, or As |
| energy_keV | float | Implantation energy (keV) |
| angle_deg | float | Beam tilt angle (degrees) |
| thickness_A | float | Substrate layer thickness (Å) |
| Rp_A | float | Projected range (Å) |
| dRp_A | float | Longitudinal straggle (Å) |
| lateral_range_A | float | Lateral range (Å) |
| lateral_straggle_A | float | Lateral straggle (Å) |
| radial_range_A | float | Radial range (Å) |
| radial_straggle_A | float | Radial straggle (Å) |
| vacancies_per_ion | float | Displacement damage vacancies per ion |
| backscattered | float | Backscattered ions (out of 10,000) |
| transmitted | float | Transmitted ions (out of 10,000) |
| is_profile | int | 1 = stopped (Rp ≥ 10 Å); 0 = transmitted |

All SRIM simulations used 10,000 ions per run in TRIM full-cascade mode with amorphous single-layer targets.

## Reproducing the Full Dataset

The complete 5,611-simulation training corpus can be reproduced by:

1. Installing SRIM-2013 (http://www.srim.org)
2. Running `train_model.py` with the parameter grid defined in Section 2 of the paper:
   - 3 substrates × 5 ions × 11 energies × 6 angles × 5 thicknesses = 4,950 records (Dataset A)
   - 661 boundary-condition augmentation records (Dataset B)
